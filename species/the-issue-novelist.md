# 📝 The Issue Novelist

> *"I've written a comprehensive 3,000-word analysis of why this button should be 2 pixels to the left."*

**Scientific Name:** *Verbositus maximus*
**Conservation Status:** Common
**Habitat:** Issue trackers, discussion threads
**Threat Level to Maintainers:** 🟡 Moderate

---

## Description

The Issue Novelist is a species known for its extraordinary capacity for written output. Where most contributors file a bug report in a few sentences, the Issue Novelist produces a sprawling narrative complete with backstory, emotional arc, and occasionally a cliffhanger ending.

Their issues often contain valuable information — buried somewhere around paragraph 14. Maintainers must develop advanced speed-reading skills to extract signal from the prose. The most prolific Issue Novelists have been observed producing issues longer than some of the project's actual documentation.

The species is not inherently harmful — their thoroughness sometimes catches edge cases that terse reporters miss. The challenge is that responding to a Novel-length issue with a one-line "works as intended" can trigger a sequel.

---

## Behavioral Patterns

- Opens issues averaging 1,500+ words
- Includes detailed system specs, even for documentation issues
- Provides "context" sections longer than the bug description
- Responds to maintainer questions with follow-up essays
- Has been known to write fan fiction about the software in issue comments
- Occasionally includes ASCII art diagrams of questionable utility
