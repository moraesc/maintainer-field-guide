# Contributing to the Field Guide

Thanks for your interest in contributing to **The Open Source Maintainer's Field Guide to Digital Wildlife**! 🦎

## How to Contribute

### Adding a New Species

1. Copy the [species template](species/TEMPLATE.md)
2. Create a new file in the `species/` directory
3. Follow the naming convention: `the-[species-name].md`
4. Fill in all required sections
5. Add your species to the Species Index table in `README.md`
6. Open a pull request with a clear description

### Required Sections for New Species

Every species entry must include:
- **Common Name** and emoji
- **Scientific Name** (have fun with the Latin)
- **Conservation Status** (Endangered / Rare / Common / Abundant / Ubiquitous)
- **Habitat** (where in the repo they're found)
- **Threat Level** (🟢 Harmless / 🟡 Moderate / 🔴 High)
- **Description** (at least 2 paragraphs)
- **Behavioral Patterns** (bullet list)
- **Field Notes** (a quote or anecdote)

### Other Contributions

- **Survival tips**: Add to `survival-tips.md`
- **Glossary terms**: Add to `glossary.md`
- **Typo fixes**: Always welcome!
- **Habitat descriptions**: Expand the README

## Pull Request Guidelines

- **One species per PR** — don't bundle multiple species
- **Follow the template** — PRs missing required sections will be asked to revise
- **Keep it kind** — satire should be affectionate, not cruel
- **Descriptive PR titles** — "Add species: The Weekend Warrior" not "Update files"

## Code of Conduct

Be excellent to each other. This is a humor project, but humor should never come at someone's expense.
